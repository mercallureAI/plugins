# mercallureAI Plugins

Official Claude Code plugin marketplace for mercallureAI team.

## Usage

### Add this marketplace

```bash
/plugin marketplace add mercallureAI/plugins
```

### Install plugins

```bash
# Browse available plugins
/plugin

# Install a specific plugin
/plugin install <plugin-name>@mercallure-plugins
```

## Managing Plugins

```bash
# Update marketplace
/plugin marketplace update mercallure-plugins

# Disable a plugin
/plugin disable <plugin-name>@mercallure-plugins

# Enable a plugin
/plugin enable <plugin-name>@mercallure-plugins

# Uninstall a plugin
/plugin uninstall <plugin-name>@mercallure-plugins
```
